# Prompt Quality Checklist

Use this before sending a prompt.

## Clarity

- [ ] Is the goal specific?
- [ ] Is the expected output format stated?
- [ ] Is the audience clear?
- [ ] Are important constraints included?

## Context

- [ ] Did I include relevant background?
- [ ] Did I provide source material if needed?
- [ ] Did I state what the model should assume?
- [ ] Did I state what the model should not assume?

## Output Control

- [ ] Did I request a useful structure?
- [ ] Did I specify length or detail level?
- [ ] Did I include examples if style matters?
- [ ] Did I ask for trade-offs when there are multiple good answers?

## Reliability

- [ ] Did I ask the model to flag uncertainty?
- [ ] Did I ask for verification when facts may be current or high impact?
- [ ] Did I avoid asking for unsupported claims?
- [ ] Did I include a review or critique step for important work?

## Reuse

- [ ] Can this prompt be turned into a template?
- [ ] Are variable parts easy to replace?
- [ ] Does it work for more than one similar task?
