# Job Hunt Prompt Pack

## LinkedIn About Section

```text
You are a personal branding coach for frontend/UI engineers.

Goal:
Rewrite my LinkedIn About section.

Context:
- I build UI-focused web applications
- I want to highlight frontend craft, design systems, and AI-assisted workflows
- Target audience: recruiters and engineering managers

Tone:
Clear, confident, practical, not inflated.

Output:
1. Short About version
2. Longer About version
3. Headline options
```

## GitHub Project Description

```text
Write a GitHub repo description for this project:
[project summary]

Make it:
- under 160 characters
- specific
- recruiter-friendly
- technical enough for engineers

Return 10 options.
```

## Interview Story

```text
Turn this project into a STAR interview story:
[project details]

Output:
- Situation
- Task
- Action
- Result
- Follow-up questions I should prepare for
```

## Recruiter Message

```text
Draft a short recruiter message.

Context:
- I am applying for frontend/UI engineer roles
- I have a GitHub project showing UI skills and prompt engineering workflows

Constraints:
- Keep it under 900 characters
- Sound human
- Include one clear call to action
```

## Project Launch LinkedIn Post

```text
Write a LinkedIn post announcing my new GitHub project.

Project:
[project link and summary]

Tone:
Professional, concise, learning-focused.

Avoid:
- hype
- buzzword overload
- pretending the project is bigger than it is
```
