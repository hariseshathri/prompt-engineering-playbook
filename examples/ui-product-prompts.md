# UI and Product Prompt Examples

## Build a Portfolio Project

```text
You are a senior frontend engineer and product designer.

Goal:
Design a portfolio-ready UI project that proves frontend craft.

Context:
- Target role: frontend/UI engineer
- Audience: recruiters and hiring managers
- The project should feel like a real product, not a landing page

Constraints:
- Include responsive layouts
- Include interaction states
- Include accessible controls
- Include data or workflow complexity

Output:
1. Product concept
2. Core screens
3. Component list
4. Interaction details
5. README pitch
```

## Improve a Dashboard

```text
Review this dashboard idea:
[paste idea]

Evaluate:
- hierarchy
- scan speed
- data clarity
- empty/loading/error states
- mobile behavior

Return a prioritized redesign plan with concrete component changes.
```

## Turn UI Work Into Resume Proof

```text
Convert this UI project into resume bullets:
[paste project details]

Constraints:
- Make every bullet evidence-based
- Avoid vague words like "helped" or "worked on"
- Include product, engineering, and accessibility value

Return 5 options ordered from strongest to weakest.
```

## Create a Design System Checklist

```text
Create a checklist for reviewing a small design system.

Include:
- tokens
- components
- states
- accessibility
- documentation
- developer handoff

Output as a markdown checklist.
```
